Lesson One: The Basics of the Web

An Overview of the Web

The web consist of a vast amount of computers from all over the world that interconnect with each other. When someone goes to a web page, the information entered is a HTTP Request. This request is uses a HyperText Transfer Protocol(HTTP), which is sent to a server. The server replies finding the correct HTML document which the web browser interprets and displays it on the computer screen. Major Pieces of the Web

  • HTTP-The main protocol of the web
  • Servers-Computers that host the files that make uo the web
  • Internet-The world's largest computer network
  • Browser-A program that runs on your computer to display files found on the web

    HTML

    HTML is an acronym for HyperText Markup Language and in essence is heart of the web. HTML is composed of:

  • Text content "what you see"
  • Markup "how it looks"
  • References to other documents eg. images and videos
  • Links to other pages

    • To explain in full here is a video.

    HTML Markups

    HTML markups are made up of tags. They start with a opening tag and end with a closing tag. Inbetween the opening and closing tags is the content, this is also a tag. These tags all together as a whole are called an element.

    Some Example of Tags

  • b = Bold
  • em = Italic
  • p = Paragraph
  • br = Line break
  • img = Images

    • Elements (Inline and Block)

    There are two types of elements inline and block. Inline only occupies the space along the content. Block forms a box or block that works like a container around the content.
